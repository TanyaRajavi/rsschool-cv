code example

Task: Write a method, that will get an integer array as parameter and will process every number from this array. Return a new array with processing every number of the input-array like this: If the number has an integer square root, take this, otherwise square the number. Notes: The input array will always contain only positive numbers, and will never be empty or null. 

       
            function squareOrSquareRoot(array) {
              array.forEach((item, idx) => {
                if (Number.isInteger(Math.sqrt(item))) {
                  array[idx] = Math.sqrt(item);
                } else {
                  array[idx] *= array[idx];
                };
              });
              return array;  
            }
